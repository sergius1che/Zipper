# Zipper
10 kB utility for archiving and unzipping files and directories

Example

1. To archiving and move file: zipper C:\tmp\foo.txt C:\tmp\NewFolder\
2. To unzipping: zipper -U C:\tmp\NewFolder\foo.zip
